Typography

Three purposeful typefaces — each assigned a unique role. Never mix their roles.

Display / Brand

Cormorant Garamond

Editorial serif. Used for brand name, page titles, empty state heading. Conveys warmth and intelligence.

UI Body

Outfit

Geometric, warm sans-serif. Used for all UI labels, buttons, sidebar items. Clean without feeling cold.

Chat / Code

Fira Code

Readable mono with character. Used for all chat bubble text, code blocks, input field. Feels conversational.

Chat Bubble Component

// components/ChatBubble.jsx
import { useState } from "react";

export function ChatBubble({ role, content, time }) {
  const isUser = role === "user";
  const isError = role === "error";

  return (
    <div className={`flex flex-col gap-1 px-7 max-w-3xl mx-auto w-full
      animate-[fadeUp_240ms_ease-out_both]
      ${isUser ? "items-end" : "items-start"}`}
    >
      {/* AI header */}
      {!isUser && !isError && (
        <div className="flex items-center gap-2 mb-1">
          <div className="w-7 h-7 rounded-sm bg-accent-soft border border-accent-border
                          flex items-center justify-center
                          text-[9px] font-bold text-accent tracking-wide">
            AI
          </div>
          <span className="text-2xs font-bold uppercase tracking-widest text-accent">
            NeuralChat
          </span>
        </div>
      )}

      {/* Bubble */}
      <div className={`
        px-4 py-3 rounded-[18px] max-w-[85%] font-mono text-xs font-light
        leading-relaxed word-break transition-all duration-150
        hover:-translate-y-px hover:shadow-md
        ${isUser
          ? "bg-accent text-white rounded-br-xs shadow-btn"
          : isError
          ? "bg-danger-bg border border-danger-border text-danger rounded-tl-xs shadow-sm"
          : "bg-card border border-[rgba(90,80,60,0.12)] text-ink-primary rounded-tl-xs shadow-sm"
        }
      `}>
        {isError && <span>⚠ </span>}
        {content}
      </div>

      <span className="text-[0.67rem] text-ink-disabled font-mono px-1">
        {time}
      </span>
    </div>
  );
}

Input Bar Component

// components/InputBar.jsx
export function InputBar({ onSend, disabled }) {
  const [value, setValue] = useState("");

  return (
    <div className="px-7 py-3 pb-5 bg-surface/90 backdrop-blur-sm border-t border-[rgba(90,80,60,0.1)]">
      <div className={`
        flex items-end gap-2 bg-card border-[1.5px] rounded-[24px]
        px-4 py-2.5 max-w-3xl mx-auto transition-all duration-240
        ${value
          ? "border-accent-light shadow-focus-ring shadow-sm"
          : "border-[rgba(90,80,60,0.16)] shadow-sm"
        }
      `}>
        <textarea
          value={value}
          onChange={(e) => setValue(e.target.value)}
          placeholder="Ask anything…"
          rows={1}
          className="flex-1 font-mono text-[13px] font-light text-ink-primary
                     bg-transparent border-none outline-none resize-none
                     leading-relaxed min-h-[22px] max-h-[200px] caret-accent"
        />
        <SendButton disabled={!value.trim() || disabled} onClick={() => { onSend(value); setValue(""); }} />
      </div>
      <p className="text-center text-[0.67rem] text-ink-disabled font-mono mt-2">
        Press <kbd>Enter</kbd> to send · <kbd>Shift+Enter</kbd> for new line
      </p>
    </div>
  );
}

Sidebar History Item

// components/SidebarItem.jsx
export function SidebarItem({ session, isActive, onSelect, onRename, onDelete }) {
  const [hovered, setHovered] = useState(false);

  return (
    <li
      onClick={() => onSelect(session.id)}
      onMouseEnter={() => setHovered(true)}
      onMouseLeave={() => setHovered(false)}
      className={`
        flex items-center gap-2 px-2.5 py-2 rounded-sm cursor-pointer
        border transition-all duration-130 min-h-[36px] select-none
        ${isActive
          ? "bg-accent-soft text-accent border-accent-border translate-x-0.5"
          : "text-ink-secondary border-transparent hover:bg-[rgba(90,80,60,0.055)] hover:text-ink-primary hover:border-[rgba(90,80,60,0.1)] hover:translate-x-0.5"
        }
      `}
    >
      <ChatIcon className="w-3.5 h-3.5 opacity-40 shrink-0" />
      <span className="flex-1 truncate text-[13px]">{session.label}</span>

      {/* Action buttons — appear on hover */}
      {(hovered || isActive) && (
        <div className="flex gap-0.5">
          <ActionButton onClick={(e) => { e.stopPropagation(); onRename(session.id); }} icon="edit" />
          <ActionButton onClick={(e) => { e.stopPropagation(); onDelete(session.id); }} icon="trash" danger />
        </div>
      )}
    </li>
  );
}

Micro-Interaction Patterns

Hover Lift

Elements lift translateY(-1px) or translateY(-2px) on hover. Combined with increased shadow. Creates a sense of physicality.

transition: transform 130ms ease-out,
            box-shadow 130ms ease-out;

&:hover {
  transform: translateY(-1px);
  box-shadow: var(--shadow-md);
}

Button Glow

Primary buttons increase their warm box-shadow on hover. Glow uses the accent color at 38% opacity — never harsh.

.btn-primary:hover {
  box-shadow:
    0 4px 16px rgba(74,110,93,0.38);
  transform: translateY(-1px);
}

Message Fade-In

New messages enter with a smooth fadeUp — opacity 0→1 and translateY 8px→0. Duration 240ms. No bounce.

@keyframes fade-up {
  from {
    opacity: 0;
    transform: translateY(8px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.message-group {
  animation: fade-up 240ms
    cubic-bezier(0.16,1,0.3,1) both;
}

Focus Ring

Override the default browser blue. Use the muted accent at 16% opacity for a calm, accessible focus indicator.

:focus-visible {
  outline: 2px solid #5E8573;
  outline-offset: 2px;
  border-radius: 4px;
}

/* Input bar specific */
.input-bar:focus-within {
  border-color: #5E8573;
  box-shadow:
    0 0 0 3px rgba(74,110,93,0.16),
    0 2px 8px rgba(60,45,30,0.07);
}

Design Reasoning

Why Warm Ivory?

Pure white (#FFFFFF) creates harsh contrast that causes eye fatigue in long reading sessions. Warm ivory (#F5F2EC) reduces the effective contrast ratio by ~15% while maintaining WCAG AA compliance. Users can stay focused longer.

Why Sage Teal (not Blue)?

Every AI product uses blue/purple. Sage teal (#4A6E5D) is calm, warm, and distinctive. It reads as "intelligent" without the cold tech-startup feel. It also harmonizes naturally with the warm ivory base.

Why Cormorant Garamond?

Most SaaS products use Inter or Roboto. Cormorant Garamond is a classical serif — it signals depth, expertise, and permanence. Used only for the brand and hero headings, it creates editorial contrast with the clean Outfit body font.

Why Warm Sepia Shadows?

Standard box shadows use rgba(0,0,0,…) — pure black — which reads as cold and flat on warm backgrounds. Sepia-toned shadows rgba(60,45,30,…) integrate naturally with the warm palette, creating depth that feels cohesive, not pasted-on.
